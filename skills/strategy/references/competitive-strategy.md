# 竞争战略 (Competitive Strategy)
> 来源：《Competitive Strategy》— Michael Porter（1980）
> 延伸：《Competitive Advantage》— Michael Porter（1985）

## 一句话定位

帮你看清一个行业"钱好不好赚、竞争有多惨烈"，然后选出一个能赢的姿势。适合创业者做行业分析、管理者做战略定位、投资人看赛道。

## 核心框架

### 五力模型 (Five Forces)

分析一个行业的竞争强度和盈利潜力，看的不只是现有对手，而是五股力量的综合博弈：

| 力量 | 核心问题 | 关键指标 |
|------|---------|---------|
| **现有竞争者的竞争强度** | 行业内玩家打得多凶？ | 数量、增速、差异化程度、退出壁垒 |
| **新进入者的威胁** | 外面的人容易杀进来吗？ | 资本门槛、品牌壁垒、政策牌照、规模效应 |
| **替代品的威胁** | 用户会不会换一种方式满足需求？ | 替代品的性价比、转换成本 |
| **买方议价能力** | 客户能压你多少价？ | 客户集中度、信息透明度、转换成本 |
| **供方议价能力** | 供应商能卡你多少脖子？ | 供应商集中度、独特性、替代难度 |

**关键原则**：五力越强，行业利润越薄。不是每个行业都值得进，选赛道比选策略更重要。

### 三大通用战略 (Three Generic Strategies)

看清行业后，你有三种基本打法：

| 战略 | 核心逻辑 | 适用条件 | 风险 |
|------|---------|---------|------|
| **成本领先** | 做到行业最低成本，用价格打败所有人 | 规模大、标准化产品、价格敏感市场 | 技术变革让成本优势消失 |
| **差异化** | 做到独一无二，让客户愿意付溢价 | 有品牌/技术/服务壁垒、客户看重品质 | 差异化被模仿、溢价空间被压缩 |
| **聚焦** | 选一个细分市场，做成本领先或差异化 | 大公司看不上的小市场、特定需求群体 | 细分市场被大公司盯上 |

**关键原则**：
- 不要"卡在中间"（stuck in the middle）——既不是最便宜的，也不是最独特的，最危险
- 三种战略本质是取舍，选一个意味着放弃另外两个
- 聚焦不是"没野心"，而是"用集中兵力打局部仗"

## 怎么用（操作指南）

### 场景 A：分析一个行业值不值得进

输入：用户提供一个行业/赛道名称，想知道这个行业好不好做

执行步骤：
1. 逐一分析五力，每一力给出"强/中/弱"的判断和关键理由
2. 综合判断行业吸引力：五力整体偏强 = 行业利润薄，慎入；偏弱 = 行业利润好，可进
3. 找出最关键的 1-2 股力量——它们决定了这个行业的核心游戏规则
4. 基于分析给出建议：进不进？如果进，从哪个角度切入？

### 场景 B：为公司选择竞争战略

输入：用户有一家公司/业务，想确定竞争策略

执行步骤：
1. 先用五力快速扫描行业竞争格局
2. 评估公司当前的资源和能力——你的长板是什么？
3. 对照三大战略，判断哪种最匹配：有规模优势选成本领先，有独特能力选差异化，资源有限选聚焦
4. 检查是否"卡在中间"：如果同时追求低价和差异化，亮红灯
5. 输出战略建议，附带需要取舍的事项

### 场景 C：评估竞争对手的战略

输入：用户想看清竞争对手在干什么

执行步骤：
1. 用五力分析对手所在的竞争环境
2. 判断对手走的是哪种通用战略（成本领先/差异化/聚焦）
3. 分析对手的战略是否连贯——言行一致还是卡在中间？
4. 找出对手战略的软肋——五力中哪一力可能动摇它？
5. 输出竞争分析报告和应对建议

---

## 判断标准

- [ ] **五力分析完整** — 五股力量都覆盖了，没有遗漏
- [ ] **有优先级** — 指出了最关键的 1-2 股力量，不是平均用力
- [ ] **战略选择明确** — 清楚选了哪种通用战略，不含糊
- [ ] **取舍清晰** — 说明了选这个战略意味着放弃什么
- [ ] **没有卡在中间** — 战略建议不是"既要又要"
- [ ] **基于事实** — 每个判断都有具体的行业事实支撑，不是空谈

---

## 常见误区

1. **把五力当清单填空**。五力的价值不在于逐条填完，而在于找出决定行业命运的那 1-2 股关键力量。填完不分析，等于没做。
2. **以为差异化就是"做得好"**。差异化不是品质好，而是独特到客户愿意多付钱且对手很难模仿。品质好但可复制 ≠ 差异化。
3. **聚焦 = 做不大**。很多伟大的公司是从极度聚焦开始的。聚焦是战略选择，不是能力不足。
4. **忽略"替代品"这一力**。很多行业的真正威胁不是同行，而是客户换了一种完全不同的方式满足需求（比如外卖替代方便面）。
5. **分析完不做选择**。五力分析是手段，最终要落到"我选什么战略"这个决定上。分析得再好不选，等于零。

## 组合使用

- **竞争战略 + 蓝海战略**：Porter 帮你看清红海的格局，蓝海帮你找到跳出红海的路径。先用五力确认现有竞争是否值得打，如果五力都很强，考虑用蓝海的价值创新跳出去。
- **竞争战略 + 好战略坏战略**：Porter 提供分析框架，Rumelt 帮你把分析转化为连贯的战略行动。五力分析对应 Rumelt 的"诊断"环节。
- **竞争战略 + 战略选择级联**：Porter 回答"行业长什么样"，Playing to Win 回答"我具体怎么选"。先 Porter 再 Lafley，从行业分析到战略落地。

## Agent 指令

### 信息收集
在执行前需要知道：
- 分析的是哪个行业/赛道？（具体到细分领域，比如"新能源汽车"而不只是"汽车"）
- 用户的身份——是想进入这个行业，还是已经在里面？
- 如果已经在行业里，公司的核心能力和资源是什么？
- 主要竞争对手是谁？（如果用户能说出来）
- 分析的目的——做投资决策？战略规划？竞品分析？

如果用户没有主动提供，先问行业和目的，其他信息在分析过程中逐步确认。

### 输出格式

**行业分析**输出：
```
🔍 [行业名] 五力分析

1️⃣ 现有竞争：[强/中/弱] — [一句话理由]
2️⃣ 新进入者：[强/中/弱] — [一句话理由]
3️⃣ 替代品：[强/中/弱] — [一句话理由]
4️⃣ 买方议价：[强/中/弱] — [一句话理由]
5️⃣ 供方议价：[强/中/弱] — [一句话理由]

📊 综合判断：[行业吸引力评级 + 关键驱动力]

💡 建议：[进不进 + 如果进从哪切]
```

**战略选择**输出：
```
🎯 [公司名] 竞争战略建议

📍 行业格局：[五力快速总结]
🏢 核心能力：[公司长板]
⚔️ 推荐战略：[成本领先/差异化/聚焦] + [理由]
❌ 需要放弃：[这个选择意味着不做什么]
⚠️ 关键风险：[这个战略最大的威胁]
📋 下一步：[3个具体行动]
```

### 注意事项
- 五力分析需要行业事实支撑，如果用户给的信息不够，要主动搜索补充
- 不要给"既要又要"的建议，战略的本质是取舍
- 语气要像一个懂行的战略顾问，直接、有判断、不装腔
- 对于极度复杂的行业（如平台型企业），五力模型有局限，要诚实说明
