# 掰不弯的谈判 (Never Split the Difference)
> 来源：《Never Split the Difference》— Chris Voss（2016）

## 一句话定位

FBI 人质谈判专家的谈判方法论——用"战术同理心"而非逻辑说服来赢得谈判。适合任何需要谈判的场景：商务议价、薪资谈判、合同协商、冲突调解。

## 核心框架

### 战术同理心 (Tactical Empathy)

核心理念：谈判不是说服对方接受你的逻辑，而是让对方感觉被理解，从而愿意配合。

关键工具：

| 工具 | 做什么 | 为什么有效 | 示例 |
|------|--------|-----------|------|
| **镜像 (Mirroring)** | 重复对方最后几个关键词 | 让对方继续说下去，暴露更多信息 | 对方："我们预算有限。" 你："预算有限？" |
| **标注 (Labeling)** | 说出对方的情绪/感受 | 化解负面情绪，强化正面情绪 | "听起来你对这个时间表感到压力很大。" |
| **沉默 (Dynamic Silence)** | 说完标注或镜像后闭嘴 | 给对方空间思考和回应，避免自己说多了 | （标注后停顿至少 4 秒） |

### 校准问题 (Calibrated Questions)

用"怎么"和"什么"开头的开放式问题，把你的问题变成对方的问题：

| 问题类型 | 目的 | 经典用法 |
|---------|------|---------|
| **"我怎么才能做到呢？"** | 温和地说"不"而不破坏关系 | 对方提出不合理要求时 |
| **"什么原因导致了这个情况？"** | 让对方解释而非防御 | 需要理解对方立场时 |
| **"这个方案怎么对你的团队有利？"** | 让对方自己论证你的方案 | 需要对方内部推动时 |
| **"如果我们做了X，会发生什么？"** | 让对方思考后果 | 需要对方看到风险时 |

**关键原则**：
- 永远不要问"为什么"——"为什么"让人防御性地找理由
- 校准问题的力量在于它把问题的重量转移给对方，同时你保持了礼貌
- 你不是在问问题，你是在"用问题发号施令"

### 获取"没错" (That's Right)

- 目标：让对方说出"没错"（That's right），这意味着他们感觉被完全理解了
- 方法：用"标注"和"总结"组合——先标注对方的情绪，再总结对方的立场，让对方自己说"没错，就是这样"
- **区分**："没错"（That's right）是好的，说明对方感到被理解了；"你说得对"（You're right）是坏的，说明对方想让你闭嘴

### 锚定与让步策略

Ackerman 议价模型：
1. 设定你的目标价
2. 第一次出价：目标价的 65%
3. 逐次提高到 85%、95%、100%
4. 每次让步幅度递减，传达"我在掏老底了"的信号
5. 最后一次让步用精确数字（如 47,850 而非 48,000），精确数字暗示经过了精密计算

### 黑天鹅 (Black Swans)

每次谈判中都有 3-5 个你不知道的关键信息。这些信息一旦发现，可以改变整个谈判局面。

找黑天鹅的方法：
- 多用镜像和标注让对方多说
- 留意对方"不小心说漏嘴"的信息
- 关注对方的非理性行为——它背后一定有你不知道的原因

## 怎么用（操作指南）

### 场景 A：准备一场重要的商务谈判

输入：用户即将进行一场商务谈判（合同、价格、合作条款等）

执行步骤：
1. 明确谈判目标——你的理想结果、可接受结果、底线分别是什么？
2. 分析对手——对方的利益、压力、约束是什么？他们的"没错"可能是什么？
3. 准备标注清单——预判对方可能的情绪和顾虑，提前写好标注句式
4. 准备校准问题——对方可能提出的不合理要求，你怎么用"怎么"开头的问题化解
5. 如果涉及价格，设计 Ackerman 议价计划
6. 列出可能的黑天鹅——你不知道但需要知道的关键信息
7. 输出谈判准备清单

### 场景 B：薪资/报酬谈判

输入：用户需要谈薪资、加薪、或项目报酬

执行步骤：
1. 了解用户的目标数字和底线
2. 帮用户调研市场行情作为锚定参考
3. 设计谈判开场——先用标注建立同理心（"我知道预算总是紧张的……"）
4. 如果需要先报价，用 Ackerman 策略设计报价阶梯
5. 如果对方先报价（低于预期），设计校准问题回应
6. 准备"如果不能加薪"的替代方案谈判策略
7. 输出谈判脚本

### 场景 C：处理客户投诉/冲突调解

输入：用户面临客户投诉或团队内部冲突需要化解

执行步骤：
1. 了解冲突情况——发生了什么？对方的诉求和情绪是什么？
2. 设计标注策略——先标注对方的情绪和感受，不急于解释或辩护
3. 目标：获得对方的"没错"——让对方感到被完全理解
4. 用校准问题引导对方参与解决方案设计——"你觉得什么方案对你是公平的？"
5. 设计让步策略——如果需要让步，让步要有节奏、有理由
6. 输出冲突处理方案

## 判断标准

- [ ] **先理解后表达** — 在提出自己的方案之前，先确保对方感到被理解
- [ ] **用了标注** — 主动识别并说出了对方的情绪和感受
- [ ] **问题是"怎么/什么"型** — 没有用"为什么"引发防御，也没有用封闭式问题
- [ ] **有"没错"时刻** — 在关键节点让对方说出"没错"
- [ ] **让步有节奏** — 不是一次性让到底，而是递减让步
- [ ] **找到了黑天鹅** — 发现了对方没有主动说但影响很大的信息

## 常见误区

1. **把"同理心"理解为"同意"**。标注对方的情绪不等于你同意对方的立场。"听起来你觉得这个价格太高了"——这是标注，不是接受对方的砍价。
2. **在情绪问题上讲逻辑**。对方在生气的时候跟他讲道理没有用。先标注情绪（"你显然对这个结果很失望"），等对方情绪缓解了再谈方案。
3. **"折中"看起来公平实际上双输**。永远不要折中（Split the Difference）——折中不是好的谈判结果，是双方都没尽力的偷懒。
4. **怕沉默**。标注或镜像之后，很多人因为不舒服会自己打破沉默——然后说太多。4 秒沉默是你最好的武器。
5. **准备不够就上场**。Voss 的方法需要提前准备标注、校准问题、议价阶梯。即兴发挥在谈判中通常意味着失败。

## 组合使用

- **谈判术 + SPIN 销售**：SPIN 在销售中建立需求，Voss 的方法在谈判阶段（价格、条款）使用。SPIN 解决"客户为什么要买"，Voss 解决"以什么条件买"。
- **谈判术 + 挑战者销售**：挑战者的"控制"环节可以直接借用 Voss 的校准问题和锚定策略。当客户想压价或拖延时，用"我怎么才能做到呢？"比直接拒绝更有效。
- **谈判术 + 全新销售**：Pink 的"调频"（Attunement）和 Voss 的"战术同理心"理念一致，但 Voss 提供了更具体的工具（镜像、标注、校准问题）。

## Agent 指令

### 信息收集
在执行前需要知道：
- 谈判的类型——商务合同？薪资？客户投诉？合作条款？
- 你的目标是什么？理想结果、可接受结果、底线？
- 对方是谁？他们的利益、压力、约束？
- 你和对方的关系——初次接触还是长期合作？
- 有没有已知的"红线"——对方绝对不能接受什么？

先问谈判类型和目标，其他在准备过程中逐步挖掘。

### 输出格式

**谈判准备清单**输出：
```
🤝 [谈判场景] 谈判准备

🎯 目标设定：
- 理想结果：[……]
- 可接受：[……]
- 底线：[……]

🧠 对手分析：
- 对方利益：[……]
- 对方压力：[……]
- 对方的"没错"：[对方最想被理解的是什么]

🏷️ 标注清单：
1. "听起来你……"
2. "看起来……对你来说很重要"
3. "你似乎担心……"

❓ 校准问题库：
1. "我们怎么才能让这个方案对双方都有效？"
2. "什么因素对你来说最关键？"
3. "如果……会发生什么？"

💰 Ackerman 议价（如涉及价格）：
- 第一次报价：[目标价 × 65%]
- 第二轮：[× 85%]
- 第三轮：[× 95%]
- 最终：[目标价，精确数字]

🦢 黑天鹅猜测：
- [可能存在的未知信息1]
- [可能存在的未知信息2]
```

### 注意事项
- 谈判准备比谈判本身更重要——投入 80% 的时间在准备上
- 标注是最高频使用的工具——帮用户多准备几个标注句式
- 校准问题要自然，不能像在念剧本
- 在薪资谈判场景中，要帮用户保持尊严和职业态度，不是教用户耍手段
- 这套方法适用于高利害谈判。日常小事不需要 Ackerman 议价
