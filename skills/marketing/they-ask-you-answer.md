# 内容营销飞轮
> 来源：《They Ask, You Answer》— Marcus Sheridan（2017/2019修订版）
> 延伸：《Content Inc.》— Joe Pulizzi（2015）

## 一句话定位

最好的营销不是"推销"，而是"回答"。当你成为行业里最愿意诚实回答用户问题的那个品牌，信任和流量自然来。

## 核心框架

### 核心逻辑

用户在做购买决策之前，会先搜索大量问题。谁回答了这些问题，谁就赢得了信任——信任转化为生意。

**大多数品牌的做法**：回避敏感问题（价格、缺点、竞品对比），只讲自己好。
**They Ask You Answer 的做法**：用户问什么，你就诚实回答什么——包括尴尬的问题。

### 五大必写内容类型（The Big 5）

这五类内容是用户购买前搜索量最大的，也是大多数品牌不愿意写的：

**1. 价格与费用（Pricing）**
- 用户最想知道但品牌最不想说的
- 不是列一个价格表，而是诚实讨论影响价格的因素
- 即使你的定价复杂，也可以写"影响XX价格的5个因素"
- 为什么有效：当别人都藏着价格，你的透明就是最大的差异化

**2. 问题与缺点（Problems）**
- 诚实讨论你的产品/行业的不足、风险、不适合的人群
- "XX产品的5个常见问题"或"XX不适合哪些人"
- 为什么有效：主动说缺点 = 极度可信。用户心想"连缺点都告诉我，其他内容肯定也是真的"

**3. 对比（Versus/Comparisons）**
- 用户在做决策时一定会比较。你不写，他们去别处找——你失去了影响决策的机会
- "A vs B：哪个更适合你？"格式
- 关键：公正客观，不踩竞品。最好的态度是"帮你做出最适合你的选择"
- 为什么有效：做公正的对比 → 被视为行业专家 → 信任

**4. 评测与排名（Reviews/Best of）**
- "2024年最好的5款XX"、"XX产品评测"
- 即使你的产品不排第一也没关系——诚实建立信任
- 为什么有效：这类内容搜索量巨大，且用户在阅读时已经接近购买决策

**5. "什么是"类内容（What is）**
- 用户对行业术语、概念、流程的基础性问题
- "什么是XX""XX是怎么工作的""XX和YY有什么区别"
- 为什么有效：教育用户 → 建立权威 → 用户自然选你

### 内容飞轮运转逻辑

```
诚实回答用户问题 → 搜索引擎排名上升 → 更多用户找到你 
→ 建立信任和权威 → 转化为咨询/购买 → 收集更多用户问题 → 继续回答
```

这是一个越转越快的飞轮——内容越多，流量越大，信任越强，生意越好。

## 怎么用（操作指南）

### 场景 A：内容策略规划——"不知道该写什么内容"

输入：用户有产品/业务，想做内容营销但不知道从何开始

执行步骤：
1. 了解用户的业务类型和目标客户
2. 用 Big 5 框架生成内容选题：
   - 收集用户（或用户的客户）最常问的问题
   - 按 Big 5 分类——价格、问题、对比、评测、基础概念
   - 为每个分类生成 3-5 个具体选题
3. 排优先级：先写那些搜索量大、竞品没写、与购买决策直接相关的
4. 输出一份带优先级的内容日历

### 场景 B：信任建设——"用户不信任我们"

输入：用户的品牌缺乏信任感，转化靠低价或促销

执行步骤：
1. 诊断信任缺失的原因——通常是"不够透明"
2. 建议从最难但最有效的内容开始：
   - 公开讨论价格
   - 承认产品不适合谁
   - 做竞品的公正对比
3. 强调：短期内可能不舒服（竞品会看到你的内容），但长期会建立最强的品牌壁垒

### 场景 C：SEO 和内容获客

输入：用户想通过内容获取有机流量

执行步骤：
1. 用 Big 5 框架识别高搜索意图的内容方向
2. 每个选题做搜索意图分析：搜这个词的人在购买旅程的哪个阶段？
3. 为每篇内容设计转化路径：内容→ CTA → 下一步
4. 建议内容更新频率和节奏

## 判断标准

一个好的"They Ask You Answer"内容策略应该：
- [ ] **覆盖 Big 5 所有类型**——尤其是价格和问题（大多数人跳过的）
- [ ] **选题来自真实的用户问题**，不是自己拍脑袋
- [ ] **诚实到让自己有点不舒服**——如果你写的时候完全没有犹豫，可能还不够诚实
- [ ] **每篇内容都指向一个明确的下一步动作**（CTA）
- [ ] **长期坚持**——飞轮需要时间才能转起来，不是发三篇就有效果

---

## 常见误区

1. **只写"安全"的内容**。只发产品介绍和行业资讯，回避价格、缺点、竞品对比——这不是内容营销，是广告
2. **以为内容=推文/短视频**。They Ask You Answer 的核心是深度长内容（博客、指南、视频教程），不是碎片化内容
3. **不敢提竞品**。很多品牌认为提到竞品就是给对手打广告。实际上，你不提，用户也会搜——不如由你来提供公正的对比
4. **期望立即见效**。内容飞轮需要 6-12 个月才能显现效果。很多品牌写了两个月没效果就放弃了
5. **内容和销售脱节**。好的内容不只是"涨粉"——销售团队应该能把这些内容用于客户跟进。内容 = 销售工具

---

## 组合使用

- **They Ask You Answer + 定位理论**：定位决定你要在哪个领域建立权威，内容飞轮是建立权威的执行方法
- **They Ask You Answer + SUCCESs**：内容选题来自 They Ask You Answer，每篇内容的写法用 SUCCESs 优化粘性
- **They Ask You Answer + STEPPS**：回答用户问题的内容本身就有高 Practical Value（实用价值），再叠加 STEPPS 的其他维度增强传播

## Agent 指令

### 信息收集
- 用户的业务类型
- 目标客户是谁
- 目标客户在购买前通常会问哪些问题（如果用户不知道，帮他们想）
- 当前的内容策略（如果有）

### 输出格式

**内容策略**输出：

### 注意事项
- 这个框架最适合 B2B 企业和高考虑度的 B2C 产品（教育、医疗、金融、SaaS）。低考虑度的快消品用处有限
- 核心洞察是"诚实是最好的营销策略"——要帮用户克服"说出去会不会不好"的心理障碍
- 生成选题时，从用户的客户角度思考——他们在搜索什么？在担心什么？在比较什么？
- 如果用户的行业是信息不对称严重的（装修、医疗、法律、保险），这个框架的威力最大
