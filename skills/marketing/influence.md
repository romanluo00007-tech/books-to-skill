# 六大影响力原则
> 来源：《Influence: The Psychology of Persuasion》— Robert Cialdini（1984）
> 延伸：《Pre-Suasion》— Robert Cialdini（2016）

## 一句话定位

说服不是靠话术，是靠触发人脑中已有的心理开关。Cialdini 找到了 6 个最强的开关——理解它们，你就能设计更高转化的营销动作。

## 核心框架

### 六大原则

**1. 互惠（Reciprocity）**
人收到好处后会本能地想回报。
- 机制：先给予，再请求。赠品、免费试用、有价值的内容——都是在激活互惠本能
- 关键：给的东西要有真实价值，且要让对方感知到这是"额外的"
- 应用：免费样品、超出预期的服务、先提供有价值的内容再推销
- 注意：互惠不是贿赂。给的东西要和你的产品/服务相关，这样回报的路径自然指向购买

**2. 承诺与一致性（Commitment & Consistency）**
人一旦做出选择或表态，就会努力保持一致。
- 机制：让用户先做小承诺，后续大行动就顺理成章
- 关键：承诺要"主动的、公开的、付出努力的"——被动同意不够
- 应用：先让用户注册免费版（小承诺），再引导升级付费版；让用户填写问卷/参与调查（自我表态）；试用期、7天挑战
- 经典：脚在门槛里技术——先答应小请求，更可能答应大请求

**3. 社会认同（Social Proof）**
不确定时，人会看别人怎么做。
- 机制：当我们不知道怎么选时，会参考"和我相似的人"怎么做
- 关键：相似性比数量更重要。"100万用户"不如"你所在行业的300家公司在用"
- 应用：用户评价、案例研究、"最受欢迎"标签、实时购买通知、KOL推荐
- 注意：社会认同在不确定性高的时候最有效。如果用户已经很确定自己要什么，社会认同的影响力会下降

**4. 喜好（Liking）**
人更容易被自己喜欢的人/品牌说服。
- 喜好的来源：
  - 外表吸引力（设计感、视觉品质）
  - 相似性（"我们跟你一样"）
  - 赞美（"你问了一个好问题"）
  - 熟悉感（重复曝光）
  - 关联（和正面事物/人绑定）
- 应用：品牌人格化、创始人故事、社群文化、用户共创
- 注意：喜好不等于讨好。建立喜好的最强方式是"真实+共鸣"，不是"无底线迎合"

**5. 权威（Authority）**
人倾向于听从专家和权威的建议。
- 机制：权威的标志——头衔、着装、专业术语、媒体背书
- 关键：不需要是真正的顶级权威，"在这个特定领域比你懂"就够了
- 应用：专业认证、专家推荐、媒体报道引用、展示专业知识的内容
- 注意：权威+亲和=最强组合。只有权威容易产生距离感

**6. 稀缺性（Scarcity）**
越稀缺的东西人越想要。
- 机制：失去的恐惧 > 获得的欲望（损失厌恶）
- 两种稀缺：
  - 数量稀缺："仅剩 3 个名额"
  - 时间稀缺："活动今晚截止"
- 关键：稀缺必须真实。虚假稀缺被识破后信任归零
- 应用：限时优惠、限量发售、先到先得、独家资格
- 注意：稀缺性对"已经有兴趣"的人最有效——它是临门一脚，不是引起兴趣的工具

## 怎么用（操作指南）

### 场景 A：优化转化率/促销策略

输入：用户有一个产品/服务/活动，想提高转化率

执行步骤：
1. 了解用户当前的转化路径（从看到→了解→信任→购买）
2. 判断转化卡在哪个环节——不同环节适用不同的影响力原则：
   - 吸引注意：社会认同、权威
   - 建立信任：互惠、权威、社会认同
   - 推动决策：稀缺性、承诺一致性
   - 维持关系：互惠、喜好
3. 为卡点环节设计具体的策略
4. 检查策略组合——最好 2-3 个原则配合使用

### 场景 B：审查现有营销页面/话术

输入：用户有现成的销售页/产品页/推销话术

执行步骤：
1. 逐个原则扫描，标记当前用了哪些、没用哪些
2. 找出最大的缺口——通常缺少"社会认同"或"具体的行动承诺引导"
3. 给出添加建议，指明在页面/话术的哪个位置插入
4. 注意不要堆砌——每个触点用 1-2 个原则就够，太多反而让人觉得被操控

### 场景 C：设计促销活动/Campaign

输入：用户要做一场促销或营销活动

执行步骤：
1. 确定活动目标（拉新？转化？唤醒？）
2. 选择主打的 2-3 个影响力原则
3. 设计活动机制，让原则自然嵌入：
   - 互惠：活动中先给用户价值（免费课程/工具/资源）
   - 社会认同：展示参与人数、用户反馈
   - 稀缺：限时限量
   - 承诺一致性：分阶段参与（先免费报名→再深度参与→最后付费转化）
4. 输出活动方案

## 判断标准

一个说服力强的营销触点应该：
- [ ] 至少运用了 **2-3 个影响力原则**，且自然融入
- [ ] **社会认同具体可信**——有真实的用户故事/数据，不是"广受好评"
- [ ] **稀缺性真实**——虚假稀缺会反噬
- [ ] **给予在前，请求在后**——先提供价值，再提出行动要求
- [ ] **不让人觉得被操控**——如果受众能感知到"你在用技巧"，就失败了

---

## 常见误区

1. **把影响力原则当套路用**。这些原则的力量在于真实——虚假的稀缺、编造的社会认同、没有价值的"赠品"，不仅无效还会毁掉信任
2. **一次用太多原则**。堆砌所有原则在一个页面上会让用户觉得"被套路"。选 2-3 个最合适的，自然地融入
3. **忽视"喜好"原则**。很多人只关注"术"（稀缺、社会认同），忽视了最基础的——用户是否喜欢你、信任你。品牌好感度是其他原则发挥作用的基础
4. **把社会认同做成"自夸"**。"我们服务了10000家企业"是自夸，"这是XX公司用我们之后的真实反馈"是社会认同
5. **稀缺性滥用**。如果每周都"限时特惠"，用户很快就会免疫

## 组合使用

- **影响力 + 价值包装 ($100M Offers)**：影响力原则解决"怎么让人更想买"，Offers 解决"买什么、值不值"。先用 Offers 设计不可抗拒的报价，再用影响力原则优化转化路径
- **影响力 + STEPPS**：影响力中的社会认同和 STEPPS 中的 Public 维度高度相关；互惠和 Practical Value 相关
- **影响力 + SUCCESs**：SUCCESs 的 Credible（可信）维度可以用权威和社会认同来强化

## Agent 指令

### 信息收集
- 用户的产品/服务是什么
- 当前的转化路径和卡点
- 已有的营销材料（如果有）
- 目标受众的特征——他们做购买决策时最在意什么

### 输出格式

**转化优化**输出：
```
🎯 影响力策略分析

当前转化路径：[描述]
主要卡点：[描述]

策略建议：
1. [原则名称（中英文）]：[具体做法]
   放在哪里：[页面/流程的什么位置]
   
2. [原则名称（中英文）]：[具体做法]
   放在哪里：[页面/流程的什么位置]

预期效果：[简要说明]
```

### 注意事项
- 永远强调真实性。如果用户要求做"假的"稀缺或编造社会认同，要明确告知这不仅不道德，而且长期看会适得其反
- 分析时不要列出所有六个原则然后每个说一遍——要针对用户的具体场景挑最有效的 2-3 个深入讲
- 这些原则本身不分好坏，关键是用于什么目的。帮用户合理运用，但不协助做欺骗性营销
